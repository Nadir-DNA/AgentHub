# Skill : Veille réglementaire

## Quand utiliser ce skill
- Question sur une nouvelle loi
- Demande de mise à jour réglementaire
- Alerte changement URSSAF, impôts, code du travail
- Conformité RGPD

## Workflow hebdomadaire
1. Consulter les sources :
   - Journal Officiel (JO)
   - Site URSSAF
   - Site impôts.gouv.fr
   - Site CNIL (RGPD)
   - Actualités des syndicats professionnels
2. Identifier les changements pertinents pour TPE/PME
3. Résumer en 2-3 points actionnables
4. Proposer des actions correctives si nécessaire

## Points de vigilance
- **Janvier** : Changements fiscaux (loi de finances)
- **Mai** : Évolution RGPD / CNIL
- **Juillet** : Changements code du travail
- **Septembre** : Évolution URSSAF
- En continu : Arrêtés ministériels par secteur d'activité

## Pitfalls
- Ne pas alerter pour rien (taux d'intérêt qui change de 0.1%, ce n'est pas une alerte)
- Distinguer ce qui est applicable aux TPE (<10 salariés) vs PME (<250)
- Vérifier la date d'entrée en vigueur (ne pas stresser pour du futur lointain)
- Citer les sources (numéro de JO, article de loi, date de publication)
