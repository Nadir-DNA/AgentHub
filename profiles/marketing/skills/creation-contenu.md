# Skill : Création contenu réseaux sociaux

## Quand utiliser ce skill
- Demande de création de post
- Planning de contenu hebdomadaire
- Idées de stories
- Analyse des performances

## Workflow création post
1. Demander le thème/objectif :
   - Promotion (nouveau service, offre spéciale)
   - Éducatif (conseils, astuces)
   - Behind the scenes (coulisses, équipe)
   - Témoignage client
   - Événement (salon, fête, actualité)
2. Adapter selon la plateforme :
   - **Instagram** : visuel + texte court (150 caractères) + hashtags
   - **LinkedIn** : texte long + professionnel + call-to-action
   - **Facebook** : texte moyen + visuel + lien
   - **TikTok** : script vidéo court (15-60 sec)
3. Proposer le contenu au client
4. Attendre validation avant publication

## Template post Instagram (promotion)
```
📸 [Image : {description_visuel}]

{accroche_courte}

{texte_principal_2_3_lignes}

💬 Et vous, {question_engagement} ?

#{hashtag1} #{hashtag2} #{hashtag3} #{hashtag4} #{hashtag5}
```

## Template post LinkedIn (éducatif)
```
{accroche_question}

{texte_intro_2_lignes}

👉 {point_1}
👉 {point_2}
👉 {point_3}

{conclusion_call_to_action}

{hashtags_professionnels}
```

## Idées de contenu par métier

### Coiffure / Beauté
- Avant/Après transformations
- Conseils entretien cheveux/peau
- Présentation de l'équipe
- Coulisses du salon
- Témoignages clients

### Avocat / Juridique
- Veille juridique simplifiée
- Cas pratiques (anonymisés)
- Réponses aux questions fréquentes
- Actualités du droit

### Garage / Auto
- Conseils entretien véhicule
- Avant/Après réparations
- Présentation des équipements
- Astuces économie carburant

## Planning contenu type (hebdomadaire)
- **Lundi** : Post motivation / citation
- **Mardi** : Contenu éducatif (conseils)
- **Mercredi** : Behind the scenes
- **Jeudi** : Témoignage client
- **Vendredi** : Promotion / offre week-end
- **Samedi** : Story interactive (sondage, question)
- **Dimanche** : Repos ou contenu léger

## Pitfalls
- Ne pas poster plus de 1 fois par jour (sauf stories)
- Varier les types de contenu (pas que de la promo)
- Adapter le ton selon la plateforme
- Toujours proposer avant de publier
- Vérifier les droits d'image (pas de photos clients sans accord)
